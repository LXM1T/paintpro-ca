# 🎨 PaintPro CA

**Sistema de Gestión para Empresas de Pintura — California 2026**

App profesional para gestionar clientes, proyectos, presupuestos, inventario y empleados.

## 🚀 Stack
- Frontend: HTML + CSS + JavaScript (Vanilla)
- Base de datos: Supabase (PostgreSQL)
- Deploy: Vercel

## ✨ Módulos
- 👥 Clientes (CRUD completo)
- 🏗️ Proyectos (Kanban por estado)
- 📋 Presupuestos (Calculadora CA 2026)
- 📦 Inventario con alertas de stock
- 👷 Empleados
- 💰 Reportes financieros
- ⚙️ Configuración de tarifas
